Easy, accesible, team oriented

Manage your Github projects from the palm of your hand. Anytime, anywhere.

How does it work?

1- Select a repository from your GitHub Account

Just log into Github and we'll manage it from there. Select from your own, your organizations' and your forked repositories.

2- Import your project.

Select which project you'd like to import to your device. You are just one click away from starting to work.

3- Start playing

Move and delete cards, check up on issues and pull requests. On the bus, on the train, even in the toilet.

Better organization for a more productive you.
